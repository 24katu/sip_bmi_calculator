# sip_bmi_calculator
I developed a GUI-based BMI Calculator using Python and Tkinter that not only calculates Body Mass Index but also stores, tracks, and visualizes BMI data over time using a local database.
A Python GUI application built using Tkinter that calculates Body Mass Index (BMI), stores user data in a SQLite database, and visualizes BMI trends over time using Matplotlib.
This project provided exciting hands-on experience with GUI development, database integration, and data visualization.

Features of BMI_Calculator:
GUI-based application using Tkinter
Multiple user support
Add new users
Calculate BMI using height & weight
Automatic BMI category classification
Store BMI records in SQLite database
View BMI history
Visualize BMI trends over time
Color-coded BMI category display

BMI Categories Used:
BMI Range	Category
< 18.5	Underweight
18.5 – 24.9	Normal
25 – 29.9	Overweight
≥ 30	Obese

Technologies & Libraries:
Python
tkinter – GUI interface
sqlite3 – Database storage
matplotlib – BMI trend visualization
datetime – Timestamp management
